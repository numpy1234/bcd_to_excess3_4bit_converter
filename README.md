# BCD to Excess-3 Converter

## Description
A combinational BCD to Excess-3 code converter designed in Verilog.

## Files

- src/bcd_to_excess3.v  
  - RTL design module

- testbench/testbench.v  
  - Simulation testbench

## Simulation

Tested BCD inputs:
0000 to 1001

Example:

BCD = 0101 (5)

Excess-3 = 1000
